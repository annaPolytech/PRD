IBM ILOG CPLEX Optimization Studio V12.6.0

IBM® ILOG® CPLEX® Optimization Studio est le moyen le plus rapide de créer des modèles d'optimisation efficaces et des applications de pointe permettant de traiter les multiples problèmes de planification et d'ordonnancement. Avec son environnement de développement intégré, son langage de modélisation descriptive, ses outils intégrés et ses solveurs avancés, traitant les problèmes de programmation mathématique et les problèmes de programmation par contraintes, il prend en charge le processus complet de développement et de résolution des modèles.

CPLEX Optimization Studio comprend plusieurs composants. Certains composants ne sont pas disponibles sur toutes les plateformes. Pour en savoir plus, rendez-vous sur la page relative à la configuration système requise qui est décrite en détail ci-dessous. Chacun des composants est placé dans un sous-dossier (ou sous-répertoire pour UNIX/Linux) de l'emplacement d'installation. Les API et les connecteurs pour chaque composant se trouvent dans le sous-dossier (sous-répertoire) de ce composant. Par exemple, les connecteurs MATLAB, Excel et AMPL pour CPLEX se trouvent dans le sous-dossier (sous-répertoire) cplex.

  • opl : environnement de modélisation OPL et environnement de développement intégré (IDE)
  • cplex : optimiseurs CPLEX incluant les optimiseurs de simplexe, de barrière et mixte en nombres entiers
  • cpoptimizer : optimiseur PPC (CP Optimizer), qui utilise la technologie de programmation par contraintes
  • concert : couche de modélisation C++ à utiliser avec les optimiseurs

Autres répertoires :

  • doc : version HTML de la documentation, ainsi que des tutoriels au format PDF
  • jre : environnement d'exécution Java utilisé pour les processus d'installation et de désinstallation (sur les plateformes dépourvues d'IDE OPL)
  • license : documents de contrat de licence
  • properties : informations d'inventaire
  • Uninstall : informations nécessaires pour la désinstallation du produit

Note pour les utilisateurs AIX

Les versions d'IBM ILOG CPLEX Optimization Studio optimisées pour l'architecture Power 7 se trouvent dans les sous-répertoires power7-32_aix et power7-64_aix des répertoires des composants. Les bibliothèques et les fichiers exécutables contenus dans ces sous-répertoires fonctionnent seulement sur la plateforme Power 7. Les applications C ou C++ créées avec ces bibliothèques doivent utiliser les compilateurs IBM XL C/C++ for AIX, version 11.1 ou ultérieure.

Installation sous Windows

Sous Windows, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :

C:\Program Files\IBM\ILOG\CPLEX_Studio[edition]126

Pour une installation Windows 32 bits sur une plateforme Windows 64 bits, l'emplacement d'installation par défaut est :

C:\Program Files (x86)\IBM\ILOG\CPLEX_Studio[edition]126

où [édition] peut être :
* Preview Edition
* vide, pour l'édition commerciale standard
Lors de l'installation, vous avez peut-être sélectionné un emplacement différent ([rép_install]).

IDE

où [édition] peut être :

* vide, pour l'édition commerciale standard

Installation sous UNIX/Linux

Le programme d'installation pour UNIX/Linux est un fichier exécutable. Pour lancer une installation, vous devez donner à ce fichier un droit d'exécution et vous devez l'exécuter depuis une ligne de commande. Par exemple, lancez la commande ./<nominstall>.bin où <nominstall> est le nom du programme d'installation. Vous pouvez donner l'autorisation d'exécution à un fichier avec la commande chmod u+x <nom_installation>.bin.

Sous UNIX, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :

/opt/IBM/ILOG/CPLEX_Studio[edition]126

Sous Linux, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :

/opt/ibm/ILOG/CPLEX_Studio[edition]126

où [édition] peut être :
* Preview Edition
* vide, pour l'édition commerciale standard

Lors de l'installation, vous avez peut-être sélectionné un emplacement différent ([rép_install]).

IDE

Vous pouvez lancer l'IDE depuis un terminal Linux.
1. Ouvrez une fenêtre de terminal
2. Passez dans le répertoire [rép_install]/opl/oplide
3. Entrez la commande ./oplide

Vous pouvez également entrer le chemin d'accès absolu à partir de n'importe quel emplacement du terminal :
[rép_install]/opl/oplide/oplide
